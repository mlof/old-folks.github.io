MC01: Installing Vue

Context: The aim of Monday Club is to develop some apps, learning some new code in the process and spending some time with my friend on a productive project. The initial testing app is called 'app-1'. Today I installed the software I need. The installation on Windows 10 ran into issues (I think they are related to user permissions/path, I will revisit them when I have time). I switched to Linux Ubuntu 20.04 and the install took about 10 minutes. 

# Install nodejs
$  curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
$  sudo apt-get install -y nodejs
					
# Install vue using npm (node package manager)
$  sudo npm install -g @vue/cli
					
# navigate to project folder and clone project
$  cd ~/projects/web/
$  git clone %url/app_name%
$  cd %app_name%
					
# download and install dependencies
$  npm install

# start the server
$  npm run serve
Once the server is running, it will display the IP address and port required for access.
In my particular setup I'm going to run a development server that holds all my the repositories. I'll access it over SSH from whatever machine I am using.
# install openssh if required
$  sudo apt-get install openssh-server

# check the process is running after install (look for sshd process)
$  ps aux | grep sshd

# any problems, then restart it
$  /etc/init.d/sshd restart

What is a Fab Lab and the Fab Academy?

Fab Labs

A fab lab (fabrication laboratory) is a workspace equipped with a selection of tools offering various methods of digital fabrication. The aim is to be able to 'make (almost) anything' (Gershenfeld, 2005). The concept began as an outreach project from MIT's Center for Bits and Atoms (CBA), with the first lab opening in India. There is now a network of over 1000 fab labs all around the world.

The fab lab at the University of Brighton (styled as FabLabBTN) has four major machines:

  1. an Epilog Helix laser cutter
  2. a Roland MDX-50 CNC mill
  3. a Roland CAMM-1 GX-24 vinyl cutter
  4. a Creality CR-10 Mini 3D printer
The lab is also equipped with a wide selection of hand tools, some power tools and a soldering station. The lab is based in the Heavy Engineering building at the University of Brighton and fab lab students can arrange access to a plasma cutter, a larger CNC for bigger wood projects, a drill press and various lathes.

While I have listed individual machines here, the fab lab should be considered a machine.

Fab Academy

The Fab Academy program is a fast paced, project-based exploration of making things with the tools in the fab lab. It is how the fab lab network trains instructors. The learning is distributed, as opposed to distance: The tools and processes are introduced in a weekly global lecture (14:00 GMT) delivered centrally (normally by Neil Gershenfeld) and then the local lab provides support and tuition. The rest is self-motivated exploration of the topic for the week. See the list of weekly tasks to get an idea of the schedule. The weekly project assignments are compulsory, but the complexity of the outcome is variable and unprescribed past a certain base level of understanding. This approach leads to a lot of different ideas and outcomes, very different from the nearly identical products of a step-by-step tutorial. It is a learning approach that I really enjoy.
